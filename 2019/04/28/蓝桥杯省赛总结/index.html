前言

　　真的没想到自己竟然可以在蓝桥杯省赛Java B组中拿到省级一等奖。如果说真的要我说我是靠什么拿到这个名次的话，也就是平时不断的积累和思考了。千万不要想就靠着赛前的突击拿到好成绩，那是不现实的，也希望各位学弟学妹们，无论自己的基础怎么样，踏踏实实一步一步的努力，稳扎稳打，相信定会取得不错的成绩。好，废话就到为止，下面我就来介绍一下我在蓝桥杯比赛之前的一些准备吧。

　　确定了比赛名单之后我就开始准备了，首先还是去了解各种常用的api和Java的一些语法，在这里推荐一个介绍Java语法的一个网站，写的很全面，叫菜鸟教程里面有很多Java语法，如果大家平时忘记了什么语法或者api的话可以直接在这个网站上查阅。　每天都在蓝桥杯练习系统上从入门题开始刷题，简单题一天刷个五六题问题应该不大，一些涉及到算法的题目，一天１~2 题也可以，有时候很多题目不会写可以直接在百度上搜索，百度上关于算法的题解还是非常多的，很多博客都有题解，但是大家再看完这些文章之后一定要自己理解清楚原理然后自己独立把代码敲出来，切记不要 copy，　paste ，只有这样大家才能真正地提高自己的算法水平，如果有些题目实在想不明白也可以去问问老师　，我的指导老师罗恒洋就为我解答了很多问题的答案，所以大家也千万不要忘了还有我们的老师可以帮助我们。

　　大家主要准备的时间应该是寒假了，我就在寒假期间刷了很多的题目看了一些算法教程，就在蓝桥杯官方练习系统上做的，就一题一题按照顺序去做就可以了，主要就是训练自己的算法水平。寒假结束来到学校之后，我又把Java的一些常用类的操作api熟悉了一遍，不得不去强调掌握常用的类的api真的很重要，后来在考场上也着实证明了掌握了这些api对我的比赛帮助很大。大家可以去网上找历届的算法原题，自己给自己定3个小时模拟考场练习，在这个过程中相信你会自己总结出不少的考场答题技巧。大家根据自己个人的情况合理的选择学习方法，我下面推荐一些资料供大家选择。

干货推荐

书籍推荐

我觉得其实Java组对于C/C++组讨巧的地方就在于Java里有很多现成的api可以直接调用，像大数类，数组类，字符串类等，有很多很好用的api操作，大家在做题时可以直接调用即可。非常的方便。而且学习Java语法对于比赛也是非常重要的，所以我觉得大家参加Java组首先必须要做的就是熟悉API和Java语法, 所以给大家推荐几本关于Java语法的书。

  • 第一行代码 Java：我用的就是这本书，书中介绍了很多Java的操作特性，这也是我买的第一本编程书，通过这本书学习了很多的Java和编程的知识，蓝桥杯比赛我也在上面学到了些Java API操作。

  • 疯狂Java讲义 第4版 李刚 ，这本书也是学习Java很不错的书籍，里面讲的很细，很通透，也非常适合学习Java语法和一些java的api操作。

  • Java编程思想 第4版 这本书算得上是Java程序员的圣经了，如果想要深入学习Java，以后想成为一名Java程序员的话，这本书是你不得不读的了。当然不推荐完全Java小白去读，最好是学过Java，看完本书后肯定会对Java有着更深层次的认识了。

  • 算法笔记：关于介绍常见的算法像递归，动态规划，贪心等的算法书用Java也是有的，但似乎没有用C/C++的丰富，而且大多不是面向于算法竞赛，更像是针对于算法面试和日常开发，所以我给大家推荐下面这本书也是我自己用过的，里面的算法方法和技巧讲 解的十分全面，把这本书介绍的算法和解题技巧学会了，不仅仅是对于你参加蓝桥杯，还有考研，PAT甲乙级考试认证，ACM等都是大有裨益的 。虽然是C语言版本的，但是相信你要是有一定的Java基础，把C语言代码转化为Java应该不是难事。强烈推荐！这本书还配套上机训练指南，也值得一看。

注意：

①以上推荐的书籍所并不是要求全部读完，根据自己的知识盲点和薄弱点有针对地去看效果可能会更好，当然如果你没有什么基础的话，系统地看一遍也是很好的。

②我推荐的所有视频和书籍和网站的标题都是带着超链接的。可以直接点击购买或者观看访问的。

③由于蓝桥杯比赛对于jdk版本有较大的限制，我这一届只可以用1.6版本后面可能会变，JDK１.6这个版本以上的一些特性，考场上是不可以用的，大家自己注意哈。

课程推荐

蓝桥杯经典例题讲解：这个课程我是参加学校的蓝桥杯选拔之前看的，视频本身就是用Java讲的，非常适合我们Java组的同学哈，感觉老师讲解也是非常不错的，尤其是对于递归的讲解很详细，后面还有其他类型的题目老师讲解的也都非常不错，值得一看。

算法很美 : 这个是蓝桥杯官网最新推出的视频辅导教程，也是我寒假期间看的，正版的课程里每章还配套练习题和答疑课程，标题的链接是Ｂ站的，大家可以直接看看(也不知道你们看到的时候会不会被和谐了)，基本全面介绍了常用的算法，但是大家看这个课程要注意，不要以为自己看老师写了自己就会了，一定要把老师课程中的代码自己独立码出来，这样才能内化成自己的东西。

蓝桥杯官网特训：这个课程是以前蓝桥杯推出的算法辅导课程，有时间同学可以看看，有题目有视频，很系统。标题给的是Ｂ站的链接，我给个网盘链接里面有题目， 点击我，提取码: tjvn 。如果链接失效大家也可以自己去找或者去咸鱼上购买哈。

昵称五个字cg的蓝桥特训教程：这个教程是B站的一个叫昵称五个字cg的up主自己录制的，我在考试前２周看的　他的教程，我个人感觉这个up讲的教程还是非常清晰的，所以也在这里推荐给大家。

蓝桥历届真题解析－Java B组 : 这个真的是历届的赛题了(当然第10届没有哈，因为我们比赛就是第10届)，我在赛前也看了几套，大家可以配合我在上文中所说的在模拟考场练习之后去看自己不会的题目是怎么写的，这个课程里给了历届赛题和题解。大家可以几个人差着买哈。

OJ网站

蓝桥杯练习系统) : 这个我不必多说了，大家自己多练习就好，多刷题才是王道，大家尽量把题目做的全对哈。

Ｃ语言网 : 这个网站上也有蓝桥杯的题目，而且之所以推荐是因为这个网站上有很多关于题目的讨论，大家如果题目不会可以去看一下其他同学的题解。会的题目也可以去看看别人的思路，这样自己才成长的快嘛。

洛谷：这个OJ网站也非常不错有很多难度更大的题目，这个网站更加针对的是NOIP，里面还有试炼场可以一步一步闯关提高自己的算法水平。我觉得这个对于一些基础不好的同学可能意义更大的，一步一步去提升自己的算法水平。

考场技巧

其实大家在平时扎实地提升自己的算法水平考试一定没问题，考场技巧只是帮助大家更好的拿分罢了。

  • 看了历届的算法题，你会发现蓝桥杯在第一题很喜欢考日期时间题，所以要么你要学会在Excel里如何处理日期，我更推荐你去掌握Java里的关于日期和时间的一些类　———　Calendar和Date　，掌握这２个类常用的api操作。自己尝试把历届的赛题中的涉及到时间的题目都自己用这２个类去用程序做一下。
  • 蓝桥杯很多题目都是可以用暴力法解决的，尤其是填空题，大家可以直接用暴力法求解，不需要想的太复杂。平时训练的时候也记得如果自己写出来就暴力求解。当然，有些题目暴力求解也并不是很简单哈。

  • 蓝桥杯的题目不一定都要写代码来完成，合理的利用计算机里的软件，比如excel等也行会事半功倍。

  • 蓝桥杯的得分机制和一般的算法竞赛不同，他是积分制的。你不需要把所有用例都通过，你通过几个就会有多少分，在考场上时间有限，对于一些量级规模很大的数据，如果自己时间不够，该放弃就放弃，不要为了那个数据，调试了很长时间代码，等你把所有的题目都做完再去做这个更有意义一点。不要因小失大。

  • 如果一道题目实在不会写的话，你也可以把样例中的输入输出数据直接写到代码里，也许也能得几分哈。
